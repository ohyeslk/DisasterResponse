SMS can help you out of disaster.

Twilio is used for a message receiving number 213-297-3661.

Three message formats with different functions are implemented in this SMS.

1. Navigate from A to B

Request the navigation guides from A to B. This can be used to guide sender to the nearest emergency shelter.

2. Google content

Search content with Google and send back the first result. It's used to search for the nearest emergency shelter.

3. Tweet content

It tweets the content onto a public Twitter account. Like help sender need, place sender currently locates, etc.

Video demo of SMS application

Three functions are achieved with Twitter, Google Search and GoogleMap API. The above video shows the usages of this SMS. Twilio and Heroku are actually built on AWS.
